PASSO 2: Configurar Git Localmente

  git init
  git config --global user.name "Seu Nome"
  git config --global user.email "seu.email@example.com"
  git add .
  git commit -m "Initial commit: XGBoost temporal model with real data"
  git branch -M main
  git remote add origin https://github.com/SEU-USUARIO/passos-magicos-app.git
  git push -u origin main

PASSO 3: Verificar no GitHub
  Acesse: https://github.com/seu-usuario/passos-magicos-app
  Todos os arquivos devem estar lá!

PASSO 4: Deploy no Streamlit Cloud (Opcional)
  1. Acesse https://share.streamlit.io
  2. Clique em "New app"
  3. Selecione seu repositório
  4. Configure como app.py
  5. Deploy automático!
